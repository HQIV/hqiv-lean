\section*{Scope}
This standalone appendix is intentionally minimal and contains only the symbolic
proof-object pipeline and artifacts (exact rational certificates). It omits the
full Lean file dump.
The authoritative frozen copy of these files (and the companion Lean modules cited in the main manuscript) is the archive \PaperArtifactBundleZip, deposited on the \emph{same} Zenodo record as the paper: \PaperZenodoDeposit.
After unpacking, paths below are relative to \PaperArtifactBundleDir/ (see \path{MANIFEST.json}).

\section*{Current Lean build split}
The repository currently exposes two SO(8)-related build paths:
\begin{itemize}
\item \texttt{lake build HQIVPaperClaims}: lightweight manuscript target, using
the symbolic interface module \path{Hqiv/SO8ClosureSymbolic.lean};
\item \texttt{lake build HQIVSO8Closure}: full certified matrix-closure target
(\path{Hqiv/GeneratorsLieClosureData*.lean}, \path{Hqiv/GeneratorsLieClosure.lean},
\path{Hqiv/Algebra/SO8ClosureAbstract.lean}).
\end{itemize}
Both paths and their referenced files are included in \PaperArtifactBundleZip{}.

\section*{Exact Symbolic Certificate (New)}

\subsection*{Generator script}
Download \PaperArtifactBundleZip{} from \PaperZenodoDeposit, unpack, \texttt{cd} into \PaperArtifactBundleDir/, then run:
\begin{quote}\ttfamily\small
python3 scripts/generate\_symbolic\_lie\_closure\_certificate.py
\end{quote}
This script computes a rank-28 closure basis and all bracket coefficients exactly
as rational numbers, and writes:
\begin{quote}\ttfamily\small
artifacts/so8\_symbolic\_certificate.json
\end{quote}
